# Memory Diary Landing Page

Современная лендинг-страница для Memory Diary - веб-приложения для сохранения семейной истории.

## Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **shadcn/ui** - UI компоненты
- **Radix UI** - доступные компоненты

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
npm run build
npm start
```

## Деплой

Проект настроен для автоматического деплоя на Vercel. Vercel автоматически обнаружит Next.js проект и настроит деплой.

## Структура проекта

```
├── app/              # App Router страницы
├── components/       # React компоненты
├── lib/              # Утилиты
├── public/           # Статические файлы
└── README.md
```
